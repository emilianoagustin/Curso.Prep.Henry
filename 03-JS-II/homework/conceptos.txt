El bucle for te ofrece una forma rapida y sencilla de hacer algo repetidamente, esta compuesto por una expresion inicial que puede estar presente o no. Esa expresion puede ser compleja o incluso ahi tambien se puede declarar una variable. Luego sigue una expresion condicional que es la que determina si el bucle se sigue ejecutando, como es esto? simplemente si esta expresion es true sigue el flujo del bucle y lo frena una vez que es false. Luego sigue la intruccion o el codigo a ejecutar dentro de ese bucle. Por ultimo esta la expresion de actualizacion que se ocupa de actualizar algun valor en cada iteracion del bucle, en general se suele sumar un valor a la variable declarada en la expresion inicial.

El operador && que se conoce como AND o Y compara dos valores y devuelve true solo si los dos son true, por este motivo es que si no se cumple esta condicion siempre retorna false, este operador se ejecuta de izquieda a derecha, por ese motivo si el primero de los valores comparados es false devuelve false sin chequear el siguiente.

El operador || que se conoce como OR u O compara dos valores y devuelve true siempre y cuando algunos de los valores sea true, por este motivo es que siempre retorna true excepto cuando ambos valores sean false, este operador se ejecuta de izquierda a derecha, por este motivo si el primero de los valores comparados es true devuelve true sin chequear el siguiente.

El operador ! que se conoce como NOT o NEGACION cambia el valor booleano del valor al que este asignado, por ejemplo si tenemos un valor true, devuelve true pero si en cambio le agregamos el NOT asi !true, nos devolveria false.
